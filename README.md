# Учёт расходов (Expense Tracker) на Python

**Автор:** Вероника Борисенко

## Описание программы

Консольное приложение для учёта личных расходов на Python. Позволяет добавлять, просматривать, удалять и фильтровать записи о расходах, а также подсчитывать общую сумму расходов за указанный период с детализацией по категориям. Данные сохраняются в файл JSON и автоматически загружаются при запуске программы.

### Основные возможности

- ✅ Добавление нового расхода (сумма, категория, дата)
- ✅ Просмотр всех расходов с общей суммой
- ✅ Удаление расхода по номеру записи
- ✅ Фильтрация расходов по категории или дате
- ✅ Подсчёт суммы расходов за произвольный период
- ✅ Детализация расходов по категориям за период
- ✅ Автоматическое сохранение и загрузка данных в JSON
- ✅ Проверка корректности ввода (отрицательные суммы, формат даты)

### Технологии

- Язык: Python 3.6+
- Хранение данных: List[Expense]
- Сериализация: JSON (модуль json)
- Работа с датами: datetime
